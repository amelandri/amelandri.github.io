I've just released a small update to [JTexFileParser][1] adding the ability to export text file data to XML. [**Check it out**][2] at [google code][2].

[JTexFileParser][1] is a java package built for simplifing delimited text files parsing. It gives you a quick way for parsing a delimited text file, managing rows and row’s fields.
